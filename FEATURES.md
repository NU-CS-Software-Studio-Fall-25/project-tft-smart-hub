# TFT Smart Hub - Features

## Core Features

### User Authentication
- User registration with email and password (Youran Ma)
- Login with Google Account (Ruitao Zhou)
- Password requirements: 8+ characters, uppercase, lowercase, digit, special character (Ruitao Zhou)

### Team Compositions
- Browse TFT team compositions (System teams and User teams) (Youran Ma, Ruitao Zhou)
- View team details (Youran Ma)
- Create team composition (Youran Ma)
- Validate inputs and enforce required fields with proper prompts (Ruitao Zhou)

### Social Features
- **Comments**: Share insights and discuss team compositions (Ruitao Zhou)
- **Favorites**: Save preferred team strategies for quick access (Ruitao Zhou)
- **Likes**: Show appreciation for effective team builds (Ruitao Zhou)

### Progressive Web App (PWA)
- Install as native app on mobile and desktop (Ruitao Zhou)

### Composition Search
- Champion-based search interface to select champions and display matching team compositions (Ruitao Zhou, Youran Ma)
